**Sort the array using selection sort array** 

**INPUT FORMAT** \
Integer N = Specify the size of array which we take input from user \
Specify the input array values with space separated

**OUTPUT ARRAY** \
Sorted array

**SAMPLE INPUT** \
5
-5 -4 -3 -2 -1

**SAMPLE OUTPUT** \
-1 -2 -3 -4 -5

**TIME COMPLEXITY** \
O(n^2)
