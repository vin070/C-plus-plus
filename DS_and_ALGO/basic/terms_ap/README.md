**Write a program to print first x terms of the series 3N + 2 which are not multiples of 4.**

**INPUT FORMAT** \
Integer x

**OUTPUT FORMAT** \
Terms of series seprated by space 

**CONSTRAINTS** \
1 <= x <= 1,000

**SAMPLE INPUT** \
4

**SAMPLE OUTPUT** \
5 11 14 17

**TIME COMPLEXITY** \
O(x + number of terms divisibe by 4)
