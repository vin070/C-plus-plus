**Geometric Sum**\

Given k, find the geometric sum i.e\
1 + 1/2 + 1/4 + 1/8 + ... + 1/(2^k) using recursion.

**INPUT FORMAT**\
Integer k

**OUPUT FORMAT**\
Geometric sum (upto 5 decimal places)

**CONSTRAINTS**\
0 <= k <= 1000

**SAMPLE INPUT**\
3

**SAMPLE OUPUT**\
1.87500

**TIME COMPLEXITY**\
O(k) where k number of gp series
