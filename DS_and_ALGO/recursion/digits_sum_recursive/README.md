**Sum of digits (recursive)**\

Write a recursive function that returns the sum of the digits of a given integer.\

**INPUT FORMAT**\
Integer N

**OUTPUT FORMAT**\
Sum of digits of N

**CONSTRAINTS**\
0 <= N <= 10^9

**SAMPLE INPUT**\
12345

**SAMPLE OUTPUT**\
15

**TIME COMPLEXITY**
O(n) where n number of digits in given number
