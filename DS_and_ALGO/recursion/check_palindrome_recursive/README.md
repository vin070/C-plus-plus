**Check Palindrome (recursive)**

Check whether a given String S is a palindrome using recursion. Return true or false.

**INPUT FORMAT**\
String S

**OUTPUT FORMAT**\
'true' or 'false'

**CONSTRAINTS**\
0 <= |S| <= 1000\
where |S| represents length of string S.

**SAMPLE INPUT**\
racecar

**SAMPLE OUTPUT**\
true

**TIME COMPLEXITY**\
O(n/2) where n is length of string
