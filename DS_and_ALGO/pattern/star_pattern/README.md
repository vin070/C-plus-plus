**PRINT THE FOLLOWING PATTERN FOR GIVEN NUMBER N** \
N=4 

SSS\*SSS \
SS\*\*\*SS \
S\*\*\*\*\*S \
\*\*\*\*\*\*\* 

**INPUT FORMAT** \
N (Total number of rows)

**OUTPUT FORMAT** \
Pattern in N lines

**SAMPLE INPUT** \
3

**SAMPLE OUTPUT** \
SS\*SS \
S\*\*\*S \
\*\*\*\*\* 

**TIME COMPLEXITY** \
o(n^2)

**Note** = S represent the one spacebar.
