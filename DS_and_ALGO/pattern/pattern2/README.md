**Print the following pattern for given number N** \
Pattern for N=4 \
S S S 1 \
S S S 3 \
S 3 4 5 \
4 5 6 7 

**INPUT FORMAT:** \
N (Total number fo rows)

**OUTPUT FORMAT** \
Pattern in N lines

**SAMPLE INPUT** \
5

**SAMPLE OUTPUT** \
S S S S 1 \
S S S 2 3 \
S S 3 4 5 \
S 4 5 6 7 \
5 6 7 8 9

Note S represent the space

**TIME COMPLEXITY** \
O(n^2)
