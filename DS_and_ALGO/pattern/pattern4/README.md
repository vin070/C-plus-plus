**NUMBER PATTERN** \
Print the following pattern for the given N number of rows.

Pattern for N = 4 \
1 \
11 \
202 \
3003

**SAMPLE INPUT** \
Integer N (Total no. of rows)

**SAMPLE OUTPUT** \
Pattern in N lines

**OUTPUT FORMAT** \
Pattern in N lines

**SAMPLE INPUT** \
5

**SAMPLE OUTPUT** \
1 \
11 \
202 \
3003 \
40004

**TIME COMPLEXITY** \
O(n^2)
