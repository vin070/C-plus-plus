**Print the following number for given N number of rows** \
Pattern for N=3

A \
BB \
CCC

**INPUT FORMAT** \
Integer N (Total number of rows) 

**OUTPUT FORMAT** \
Pattern in N lines

**SAMPLE INPUT 1:** \
6

**SAMPLE OUTPUT 1:** \
A \
BB \
CCC \
DDDD \
EEEEE \
FFFFFF  

**SAMPLE INPUT 2:**
7

**SAMPLE OUTPUT 2** \
A \
BB \
CCC \
DDDD \ 
EEEEE \
FFFFFF \
GGGGGGG

**TIME COMPLEXITY** \
O(n^2)
